# Azizbek Portfolio

Modern, interactive portfolio website built with vanilla JavaScript, CSS3, and GSAP animations.

## Features

- **Custom Cursor** - Interactive cursor with follower effect
- **Particle Background** - Dynamic connected particles
- **GSAP Animations** - Smooth scroll-triggered animations
- **Glass Morphism** - Modern glass-effect UI components
- **Dark/Light Theme** - Toggle with localStorage persistence
- **Responsive Design** - Mobile-first approach
- **Skill Bars** - Animated progress indicators
- **Project Filter** - Filter projects by category
- **Contact Form** - Animated form with validation

## Technologies

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- JavaScript (ES6+)
- GSAP 3.12+ (GreenSock Animation Platform)
- ScrollTrigger Plugin

## Sections

1. **Hero** - Introduction with typing animation
2. **About** - Personal information and stats
3. **Skills** - Technical skills with progress bars
4. **Projects** - Portfolio projects with filter
5. **Contact** - Contact form and social links

## Setup

Simply open `index.html` in a browser. No build process required.

## Customization

Edit `index.html` to update:
- Personal information
- Project details
- Social links
- Skills

Edit `css/style.css` to modify:
- Color scheme (CSS variables)
- Typography
- Layout
